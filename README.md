##一、方案标题
--
    宠物喂食器

##二、方案应用场景
--
    地点：家庭、宠物医院、宠物店；
    功能：现代工作生活忙碌，不能及时照看宠物，此产品可以大幅度减少宠物对人类的依赖，定时给宠物投放实物，解决了当主人离家时，没人投喂食物的问题
